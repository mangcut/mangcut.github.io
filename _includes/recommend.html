    {% for post in site.posts reversed %} {% if post.series_id == site.recommended_series_id %} {% if page.url == post.url %}
  • {{post.subtitle}} (bài này)
    • {% else %}
  • {{ post.subtitle }}{% if post.hot %} {% endif %}
    • {% endif %} {% endif %} {% endfor %}

Có thể bạn cũng thích:

    {% for post in site.posts %} {% if post.recommend %} {% if page.url != post.url %}
  • {{ post.title }}{% if post.hot %} {% endif %}
    • {% endif %} {% endif %} {% endfor %}
